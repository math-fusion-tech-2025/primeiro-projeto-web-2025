Sobre o Projeto

O MATH<FUSION>TECH nasceu para conectar a lógica do pensamento matemático com a criatividade da tecnologia. Vamos aprender juntos a construir para a web!

Ferramentas que Vamos Usar

Estas são as quatro ferramentas que apresentamos em nosso primeiro encontro:

  • VS Code: Nosso ambiente de desenvolvimento, nosso "escritório digital".
  • GitHub: Nossa plataforma para salvar, compartilhar e colaborar em projetos.
  • HTML: A linguagem de marcação que define o esqueleto das nossas páginas.
  • CSS: A linguagem de estilização que define as roupas e o design.
